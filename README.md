## Descripción
Rediseño del sitio web corporativo de ADIRH, desarrollado como proyecto académico de la materia **Experiencia de Usuario** en la carrera de Ingeniería en Desarrollo y Gestión de Software — Universidad Tecnológica de Tecámac.

## Tecnologías
- HTML5 / CSS3 / JavaScript vanilla
- Material Design 3 (normativa de diseño)
- Fuentes locales: Wallington Pro + Montserrat
- WCAG 2.1 AA (accesibilidad)
